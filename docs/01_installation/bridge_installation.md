# IBM Cloud Pak Vault Bridge installation

## [1] Overview

### [1.1] Installation options
1. Standalone bridge (Running outside Cloud Pak cluster)
2. Kubernetes (K8s) deployment (Running in the same Cloud Pak cluster)

#### [1.1.1] Stand-alone bridge
1. Download and clone this repository
2. Generate and configure TLS key and certificate
3. Install JWT public key
4. Run bridge server
